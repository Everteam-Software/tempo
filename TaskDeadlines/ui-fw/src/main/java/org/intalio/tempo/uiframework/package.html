UI Framework component of Intalio|BPMS used as point of access for User to work with assignend Tasks. It works in couple with Forms Manger to build the Task UI. Also it uses Task Management Services to retireve Task list, get the Task details. Authentication provided by Intalio|BPMS RBAC security framework. Respective sub-packages provide classes for integration of UI Framework with aformentioned components.
